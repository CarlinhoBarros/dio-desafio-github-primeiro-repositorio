git clone <endereco> - copia o projeto do github

git status - verifica o status do projeto

git add . ou -a - adiciona as modificações a maquina local

git commit -m <mensagem> - comita a modificação com uma mensagem

git push origen main - envia todos os commits para o github

git remote set-url origin "https://(username)@github.com/(username)/(nome do repositório).git" - alterar/atualizar a origem do repositório
